# Proyecto 4 Tripleten

Homeland

## Descripción del proyecto

Crear una página web que muestre las ciudades de origen de algunos empleados de TripleTen.

## Características del proyecto

- Figma
- HTML5 semántico
- CSS
- Grid
- Flexbox
- Posicionamiento
- Iframe
- Metodología BEM
- Diseño responsivo
- Media Queries
